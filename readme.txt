
Скрипты транспилируются в ES5 с помощью babel c конфигом stage-3 плюс минифицируется и плюётся в public/js
Входной файл в src - index.js - в него подключаем необходимые зависимости

Для скриптов юзается Webpack 4 - его "конфиг" лежит в конфиге галпа в webpackStream

Все таски для удобства записаны в дефортную таску галпа "gulp"

Отдельные таски:
gulp clean - очистка папки public
gulp html - копирование основного файла в build
gulp css - сборка css стилей в build
gulp less - сборка css стилей в build
gulp scripts - сборка модулей js в build
gulp watch - отслеживание html,css/less,js изменений в исходниках
gulp webserver - поднятие dev-сервера browserSync


P.S. Инструменты, возможно, не самые оптимальные, это моя одна из первых адекватных сборок
